1. ゲーム概要

タイトル（日）：空中魔法ドッグファイト

タイトル（英）：Sky Magic Dogfight

種別：2D 上から見下ろし型 シューティング / 陣取りアクション

フォーマット：ブラウザゲーム（HTML5 Canvas + JavaScript）

プレイ人数：1人（CPU bot と対戦）
※将来的にオンライン対戦へ拡張可能な想定

2. 基本ルール

制限時間 90秒 の間に、

敵の撃墜数（KILLS）

魔法陣ゾーン滞在ポイント（ZONE）

を稼ぐゲーム。

制限時間終了でゲームオーバー → リザルト（アラート表示）

3. 操作方法
キーボード

W / A / S / D：上下左右移動

1 / 2 / 3：使用する魔法（ショット）切り替え

マウス

マウス位置：照準方向（プレイヤーの向き）

左クリック：選択中の魔法で射撃
※クリックしっぱなしで連射可能（魔法ごとにクールタイムあり）

4. プレイヤーキャラクター

外見：

「魔導書」 or 「箒」いずれかがランダムで選ばれる

表現：

常にマウス方向を向くように回転

ステータス：

HP：100

移動速度：260

被弾時：

ダメージを受け HP が 0 になると「撃墜」

約 3秒後にランダム位置でリスポーン（HP全回復）

5. 魔法（ショット）仕様

プレイヤーは 3 種類の魔法を切り替えて使用：

アーケインボルト / Arcane Bolt（キー：1）

単発の高速弾

弾速：650

ダメージ：25

クールタイム：約 0.18 秒（連射向き）

スプレッドショット / Spread Shot（キー：2）

5方向に広がる拡散弾

弾速：580

1発あたりダメージ：15

クールタイム：約 0.45 秒（面制圧向き）

ランスビーム / Lance Beam（キー：3）

直線上の高速・高威力弾

弾速：900

ダメージ：60

クールタイム：約 0.8 秒（決め技的運用）

6. 敵（Bot）の挙動

5体の Bot が常にフィールド上に存在。

挙動：

プレイヤーを追いかけない

ランダム方向へふらふら移動（一定時間ごとに移動方向をランダム変更）

壁にぶつかりそうなときは反射するように方向転換

射撃：

一定間隔ごとに、
プレイヤー付近を狙うような弾を発射

発射方向は「プレイヤー＋ランダムずれ」

HP / 撃墜：

HP100

0になると撃墜 → 約3秒後にランダムな位置でリスポーン

プレイヤーが撃墜した場合：

プレイヤーの KILLS が +1

「KILL!」のフローティングテキスト表示

7. 魔法陣ゾーン（スコアゾーン）

フィールド内に 円形ゾーン として出現。

パラメータ：

半径：約 80

出現までの待ち時間：5秒（ゲーム開始時）

ゾーンの寿命：15秒

再出現までの周期：18秒

エフェクト：

中心から外側に向かうグラデーション光

外周に回転風の輪（点線円）

内側に紋章線（放射状ライン）

効果：

キャラクター（プレイヤー & Bot）がゾーン内にいる間、

1秒あたり 1ポイント程度のペースで ZONEポイント が増加

一定間隔で「+1」の浮遊テキストが頭上に出る（臨場感）

8. スコアとゲームオーバー

KILLS：プレイヤーが敵を撃墜すると +1

ZONE：ゾーン内滞在時間に応じて加算（内部的には秒数）

ゲーム終了条件：

ゲーム時間（初期値 90秒）が 0 になると終了。

終了時：

アラートで結果を表示

日本語：

戦闘終了！ 撃墜: X 魔法陣ポイント: Y

英語：

Battle Over! Kills: X Zone Points: Y

タイトル画面へ戻る

10. 技術仕様

使用技術：

HTML5

CSS3（軽いレイアウト＆スタイル）

JavaScript（生 JS、外部ライブラリ不使用）

描画：

canvas + 2Dコンテキスト

更新：

requestAnimationFrame でループ

毎フレーム update(dt) → draw() の流れ

国際化：

texts = { ja: {...}, en: {...} } を用意
